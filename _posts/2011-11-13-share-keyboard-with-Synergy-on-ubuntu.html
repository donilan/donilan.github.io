环境 计算机1: ubuntu 11.x, 计算机名称: d-notebook, ip: 192.168.1.1 计算机2: window xp, 计算机名称: d-netbook, ip: 192.168.1.2

说明 synergy在Linux下的服务器运行名称是synergys

[bash] section: screens d-notebook: d-netbook: end

section: aliases d-netbook: 192.168.1.2 d-notebook: 192.168.1.1 end

section: links d-netbook: left = d-notebook d-notebook: right = d-netbook end

section: options screenSaverSync = false keystroke(f12) = lockCursorToScreen(toggle) endsection: screens d-notebook: d-netbook: end

section: aliases d-netbook: 192.168.1.2 d-notebook: 192.168.1.1 end

section: links d-netbook: left = d-notebook d-notebook: right = d-netbook end

section: options screenSaverSync = false keystroke(f12) = lockCursorToScreen(toggle) end [/bash]
