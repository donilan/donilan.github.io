Subeclipse, svn客户端
http://subclipse.tigris.org/update_1.6.x

JBOSS插件，很多J2EE工具
http://download.jboss.org/jbosstools/updates/stable/galileo/

Properties Editor 编辑java的属性文件，并可以自动存盘为Unicode格式
http://propedit.sourceforge.jp/eclipse/updates/

Colorer Take 为上百种类型的文件按语法着色
http://colorer.sourceforge.net/

XMLBuddy 编辑xml文件
http://www.xmlbuddy.com

Code Folding 加入多种代码折叠功能（比eclipse自带的更多）
http://www.coffee-bytes.com/servlet/PlatformSupport

RegEx Test 测试正则表达式
http://brosinski.com/stephan/archives/000028.php

Log4E Log4j插件，提供各种和Log4j相关的任务，如为方法、类添加一个logger
http://log4e.jayefem.de/index.php/Main_Page

Colorer Take 能为在Eclipse中打开的各种类型源代码文件按语法着色。支持150多种语言
http://colorer.sf.net/eclipsecolorer/

m2eclipse, maven工具
http://m2eclipse.sonatype.org/sites/m2e
