假设环境：</strong>

  1. windows系统为：windows xp</li>
  2. Window ip 地址: 192.168.1.2</li>
  3. 共享的目录名称为：share</li>
  4. 假设window下的用户名和密码为：admin/pass</li>
  5. Linux环境为:Ubuntu Linux</li>
  6. 当前在Linux下使用的用户为：doni</li> </ol> 首先我们需要安装smbfs [bash] $sudo apt-get install smbfs [/bash] 然后挂载 [bash] $sudo mount //192.168.1.2/share /path/to/your/mount/dir -o uid=doni,username=admin,password=pass,iocharset=utf8 [/bash] 下面是-o参数详解，-o是附加参数

    1. 其中uid的作用，设置挂载后用哪个linux用户为此挂载的拥有者，非常重要，如果我们用sudo挂载，一般拥有者就是root。</li>
    2. username,password这个应该不需要解析了，就是windows的用户，如果共享没有设置用户的话，可以不加这两个参数。</li>
    3. iocharset设置字符集，主要用来解决乱码问题。</li> </ol> [bash] #取消挂载 $sudo umount //192.168.1.2/share [/bash]

      另外一种方法</strong> [bash] $sudo mount -t cifs -o uid=doni //192.168.1.2/share /path/to/your/mount/dir [/bash]
