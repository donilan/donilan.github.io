常用的比变量前缀 对应的数据类型或者单词
c char或者WCHAR或者TCHAR
by BYTE(无符号字符)
n short
I int
x,y int:分别用做x坐标和y坐标
cx,cy int:分别用做x方向的长度和y方向的长度
b,f BOOL(int); f代表 falg
w WORD(无符号short)
l LONG(长整型)
dw DWORD(无符号长整型)
fn function
s string
sz string ended with zero
h handle
p pointer
cb count of bytes(字节数)
lpsz long pointer to a string ended with zero.指向以0结尾的字符串长指针。
rc rectangle(矩形)
