Vim Zen coding Plugin 安装：

  • 下载：Github
  • 安装：将下载的文件解压，然后复制到vimfiles目录或者.vim目录(如果有)下即可
  • 使用：
    1. Expand Abbreviation，输入Zen coding格式语句，然后按 Ctrl+y,然后再按","，要注意先后顺序。 Example: 输入div>p#foo$*3>a 然后按+,。 [html]

      [/html]
    2. Wrap with Abbreviation,先输入几行数据。 [text] item1 item2 item3 [/text] 然后再用选择模式"V"把要包含的数据选择上，然后再输入+,最后输入一句Zen code ul>li* [html]
      • test1
      • test2
      • test3
      [/html]
    3. Balance Tag Inward,输入+d插入模式。
    4. Balance Tag Outward,输入+D插入模式。
    5. Go to Next Edit Point,输入+n跳到下一个编辑点。
    6. Go to Previous Edit Point,输入+N跳到上一个编辑点。
    7. Update Size
      移动光标到img标签中。然后输入+i编辑属性。
    8. Merge Lines
      选择需要组合成一行的所有数据，例如以下选择所有"
    9. "标签。 [html]
      [/html] 然后输入"J"。 [html]
      [/html]
    10. Remove Tag
      [html]
      光标在这里
      [/html] 移动光标到标签块，然后输入 [/html] 然后再输入+j则连div也清除了。
    11. Split/Join Tag
      [html]
      光标放到这里
      [/html] 输入 [/html]
    12. Toggle Comment
      移动光变到块内 [html]
      Hello world!
      [/html] 输入 Hello World! --> [/html] 再次输入取消注释
    13. Make auchor from URL
      移动光标到URL [text] http://www.google.com [/text] 输入+a [html] [/html]
    14. Make quoted text from URL
      移动光标到URL [text] http://www.google.com [/text] 输入+A [html]
      Google
      ... http://www.google.com/
      [/html]
    15. Enable zencoding.vim for language you using.
      自定义你使用的语言。 {% highlight bash %} # cat >> ~/.vimrc let g:user_zen_settings = { \ 'php' : { \ 'extends' : 'html', \ 'filters' : 'c', \ }, \ 'xml' : { \ 'extends' : 'html', \ }, \ 'haml' : { \ 'extends' : 'html', \ }, \} {% endhighlight %}
