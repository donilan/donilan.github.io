废话少说，直接进入正题。

解决系统库依赖 [bash] zypper install mysql-community-server ruby libMagick++-devel libmysqld-devel libbz2-devel [/bash]

下载项目，可以到主页下载，这里是用git下载 [bash] GIT_SSL_NO_VERIFY=true git clone https://github.com/redmine/redmine [/bash]

创建mysql的数据库，因为我用mysql [bash] echo "CREATE DATABASE redmine CHARACTER SET utf8;" | mysql -uusername -ppassword [/bash] 然后修改config/database.yml文件，我想一看就明白怎么改。

安装redmine Ruby的依赖 [bash] gem install bundler

## 按自己实际情况 cd $PATH_TO_REDMINE_PROJECT

## 安装项目里的依赖，只要部署时的依赖 bundle install --without development test [/base]

一些初始化的工作 [bash] ## 生成密钥 rake generate_secret_token ## 生成数据库的表 RAILS_ENV=production rake db:migrate ## 生成一些基本的数据 RAILS_ENV=production rake redmine:load_default_data [/bash]

都没有意外的话就可以启动项目了 [bash] ruby script/rails server webrick -e production [/bash]

如果是在本地启动的话，正常来说 http://localhost:3000 就可以访问了。
